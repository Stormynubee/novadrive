# Margi Web (prototype)

Next.js mirror of the emergency flow for team demos and quick iteration. **Judges should use `novadrive-mobile/`** for APK, sensors, camera, and SMS.

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
```

Shared logic lives in `src/lib/` (START FSM, GHP, facilities). Mobile port: `../novadrive-mobile/src/lib/`.
